# Gif Browser

**Toy project using React**

## Description
Gif Browser is a small React application created as a learning project.  
It allows users to browse and search for GIFs while exploring common React patterns and frontend concepts.

## Explored Concepts
So far, this project explores:

- Props  
- `useState`  
- `useEffect`  
- Axios for HTTP requests  
- Debouncing user input  
- Adapter Pattern  
- Lifting State Up  

## Purpose
This project is intended for practice and experimentation with React fundamentals and application architecture. It is not meant for production use.

## Tech Stack
- React  
-- Axios  
-- Giphy API
